# 文件提取

#### 介绍

将此脚本所有同级目录的子目录中的某一目录里的文件copy到新生成的目录
例如目录结构是：
>资源1
>>照片
>>>1.jpg、2.jpg

>>三维模型

>资源2

>资源3

>dirSubFilesCopy.py

使用此脚本可以将照片目录中的某些文件copy到新目录，执行后如下：
>资源1
>>封面照片
>>>1.jpg

>>照片

>>>1.jpg、2.jpg

>>三维模型

>资源2

>资源3

>dirSubFilesCopy.py

其中“封面照片”为新生成的目录，1.jpg则为提取后的照片

#### 基础环境
本地安装python37
#### 使用教程
1. 将文件置于批量修改的同级目录下
2. 修改main方法中的参数
